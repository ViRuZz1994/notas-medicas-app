# 📄 Generador Profesional de Notas Médicas

Aplicación web para generar, editar y exportar notas médicas (consulta, ingreso, egreso) en formato PDF.  
Incluye plantillas rápidas de traumatología, base de datos local de pacientes, personalización de membrete y respaldo de datos.

## ✨ Características

- Tres tipos de nota: **Consulta/Evolución**, **Ingreso**, **Egreso**.
- Autocompletado de datos del paciente (guardado en `localStorage`).
- Plantillas predefinidas para fracturas, esguinces, lumbalgia, gonartrosis, etc.
- Editor en vivo con vista previa del documento final.
- Ajuste de márgenes y fondo personalizable (membrete).
- Generación de PDF limpio (tamaño carta).
- Respaldo y restauración de la base de datos local.

## 🛠️ Uso local

1. Descarga el archivo `index.html`.
2. Ábrelo con cualquier navegador moderno (Chrome, Edge, Firefox).
3. Completa los campos y genera el PDF.
